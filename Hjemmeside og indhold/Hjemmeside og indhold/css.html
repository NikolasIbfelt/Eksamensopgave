CSS

Cascading Style Sheet

Hvis HTML blev brugt til at danne skeletet for vores hjemmeside, så er CSS hvad der giver skeletet kød, blod og hud. CSS er hvad vi bruger til at style vores hjemmeside både i form af farver, og former, men også placering og styling. Vi kan bruge CSS på 3 forskellige måder som er:

  • Embedded - – Som er hvor vi skriver vores CSS stylesheet i et separat dokument for vores HTML kode, som vi linker til igennem vores HTML dokument.
  • In-line - Er hvor vi styler vores HTML kode i samme dokument. Dette bør KUN anvendes til HTML-baserede mailservere.
  • External - Er hvor vores stylesheets ligger eksternt og vi vedhjælp af internettet henter dem ned fra en ekstern server.

Det er anbefalet at skrive CSS som embedded da det bl.a. er med til at skabe bedre overblik over koden, og det er nemmere at redigere i, og kan gøres separat fra HTML koden.

Med CSS styler vi vores elementer efter vores selectors, hvor vi kan vælge om det er en bestemt class, eller et specifikt id der skal styles. Yderligere kan vi også sætte nogle regler op for elementerne der skal styles. Vi kan fx vælge at alle paragraffer der er skrevet i en div skal styles på den ene måde, mens alle unsorted lister der er skrevet i en div skal styles på en anden måde. Vi kan også pålægge vores styling ved forskellige handlinger hos brugeren, vi kan fx bruge hover til at påvirke et element til at gøre noget specifikt når musen bliver holdt over.
